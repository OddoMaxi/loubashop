# LoubaShop

**Revendeur Officiel Minelab en Afrique de l'Ouest**

Site e-commerce statique pour la vente de détecteurs d'or Minelab authentiques en Guinée, Sierra Leone, Liberia et Guinée-Bissau.

## 🌐 Live Site
https://loubashop.com (à remplacer par ton domaine)

## 📁 Structure

```
loubashop/
├── index.html          # Page d'accueil
├── product.html        # Page détail produit
├── style.css           # Styles principaux
├── product.css         # Styles page produit
├── main.js             # Logique principale
├── product.js          # Logique page produit
├── products-data.js    # Base de données produits
├── *.png               # Images produits
└── .gitignore          # Fichiers ignorés par Git
```

## 🛠 Technologies

- HTML5 sémantique
- CSS3 (Flexbox, Grid, Variables)
- JavaScript vanilla (ES6+)
- Aucun framework (site statique pur)

## 📱 Responsive

- Desktop (>1024px)
- Tablette (768px-1024px)
- Mobile (<768px)

## 🛒 Fonctionnalités

- Catalogue produits avec filtres
- Panier avec sessionStorage
- Commande via WhatsApp + Email
- Page détail produit
- Design responsive
- Animations CSS
